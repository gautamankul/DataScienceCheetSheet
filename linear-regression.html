Multiple Linear Regression & Multicollinearity

üìä What Is Multiple Linear Regression?

Multiple Linear Regression (MLR) is a statistical method used to model the relationship between a dependent variable (Y) and two or more independent variables (X‚ÇÅ, X‚ÇÇ, X‚ÇÉ...).

It estimates how changes in each independent variable affect the dependent variable.

üìê Equation
Y = Œ≤‚ÇÄ + Œ≤‚ÇÅX‚ÇÅ + Œ≤‚ÇÇX‚ÇÇ + ‚ãØ + Œ≤‚ÇôX‚Çô + Œµ
‚ùó What Is Multicollinearity?

Multicollinearity occurs when two or more independent variables are highly correlated with each other.

‚ö†Ô∏è Why is it a problem?
  • Coefficients become unreliable
  • Model becomes unstable
  • p-values become misleading
  • Hard to interpret individual variable effects
üß™ How to Detect Multicollinearity

‚úîÔ∏è Variance Inflation Factor (VIF)

VIF > 5 or 10 ‚Üí high multicollinearity.

üõ†Ô∏è How to Handle Multicollinearity

Methods include:

  1. Remove highly correlated features
  2. Use dimensionality reduction (PCA)
  3. Use regularization (Ridge/Lasso) ‚Üí recommended
  4. Combine correlated features
üßë‚Äçüíª Python Coding Example (Detect & Fix Multicollinearity)
Step 1: Import Libraries
import pandas as pd import numpy as np from sklearn.linear_model import LinearRegression from statsmodels.stats.outliers_influence import variance_inflation_factor from sklearn.preprocessing import StandardScaler
üìå Example Dataset
data = { 'X1': [10, 20, 30, 40, 50], 'X2': [11, 21, 31, 41, 51], # X2 highly correlated with X1 'X3': [5, 3, 6, 2, 1], 'Y': [100, 200, 300, 400, 500] } df = pd.DataFrame(data)
‚úÖ Step 2: Calculate VIF
X = df[['X1', 'X2', 'X3']] vif_data = pd.DataFrame() vif_data["Feature"] = X.columns vif_data["VIF"] = [variance_inflation_factor(X.values, i) for i in range(X.shape[1])] print(vif_data)

Output (example):

  Feature   VIF
0     X1  251.2
1     X2  259.4
2     X3    1.8

üëâ X1 and X2 have very high VIF, meaning they are strongly collinear.

‚úîÔ∏è Step 3A: Fix by Removing One Feature
X_new = df[['X1', 'X3']] model = LinearRegression() model.fit(X_new, df['Y']) print("Coefficients:", model.coef_)
‚úîÔ∏è Step 3B: Fix by Using Ridge Regression (recommended)
from sklearn.linear_model import Ridge ridge = Ridge(alpha=1.0) ridge.fit(X, df['Y']) print("Ridge Coefficients:", ridge.coef_)

Ridge handles multicollinearity by shrinking correlated coefficients.

‚úîÔ∏è Step 3C: Fix by PCA (Dimensionality Reduction)
from sklearn.decomposition import PCA pca = PCA(n_components=2) X_pca = pca.fit_transform(X) print(X_pca[:5])
