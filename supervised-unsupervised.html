Machine Learning Cheat Sheet

Supervised Learning

Goal: Predict known outcomes (classification or regression)

A. Regression (Continuous output)

  • Linear Regression
  • Polynomial Regression
  • Ridge / Lasso Regression
  • Support Vector Regression (SVR)
  • Decision Tree Regression
  • Random Forest Regression
  • Gradient Boosting Regression (XGBoost, LightGBM, CatBoost)

B. Classification (Categorical output)

  • Logistic Regression
  • K-Nearest Neighbors (KNN)
  • Decision Tree Classifier
  • Random Forest Classifier
  • Support Vector Machines (SVM)
  • Naive Bayes
  • Neural Networks (Feedforward, CNNs, RNNs)
  • Gradient Boosting Classifiers (XGBoost, AdaBoost, CatBoost)

Unsupervised Learning

Goal: Discover hidden structure or relationships in data

A. Clustering

  • K-Means Clustering
  • Hierarchical Clustering (Agglomerative, Divisive)
  • DBSCAN (Density-Based Clustering)
  • Mean Shift Clustering
  • Gaussian Mixture Models (GMM)

B. Dimensionality Reduction

  • Principal Component Analysis (PCA)
  • t-SNE (t-Distributed Stochastic Neighbor Embedding)
  • Autoencoders
  • Independent Component Analysis (ICA)
  • UMAP (Uniform Manifold Approximation & Projection)

D. Anomaly Detection

  • Isolation Forest
  • One-Class SVM
  • Local Outlier Factor (LOF)
